## Running the application

1. Install [Node.js](https://nodejs.org/en/)
2. ```npm install``` (inside the project folder)
3. Run ```./gulp```
4. Point your browser to [http://localhost:4000/](http://localhost:4000/)


## Running the tests

1. Run ```./gulp test```
